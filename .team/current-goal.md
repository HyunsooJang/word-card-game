# Current Goal

Use the Claude team to complete approved work with role isolation:

1. Lead owns scope, sequencing, file ownership, and final integration.
2. Requirement Analyst removes ambiguity before implementation.
3. Architect checks design, state, policy, and cross-layer risk.
4. Researcher gathers evidence without editing application code.
5. Developer implements only assigned behavior and files.
6. QA Reviewer verifies behavior, regression risk, and test coverage.

Keep work scoped, preserve existing user changes, and report verification results.
